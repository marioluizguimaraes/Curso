Os valores enviados janela do prompt é do tipo String. Para converter do tipo String para Number, posso usar os comandos:


Number.parseInt(n) ou parseInt(n) Para converter para inteiro,


Number.parseFloat(n) ou parseFloat(n) Para converter para flutuante,


Number(n) Para converção automatica,


Para converter de Number para String, usamos os seguintes comandos:


String(n) ou n.toString()
