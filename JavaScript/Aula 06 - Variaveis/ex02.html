Tipos Primitivos

  • Number: Representa valores numéricos, inteiros ou de ponto flutuante.

  • String: Representa sequências de caracteres, como texto.

  • Boolean: Representa valores lógicos verdadeiro (true) ou falso (false).

  • Null: Representa a ausência de valor ou a intencionalidade de não ter nenhum valor.

  • undefined: Representa uma variável que foi declarada, mas ainda não foi atribuída com um valor.

  • Object: Representa uma coleção de pares chave-valor e é a base para todos os outros tipos de dados em JavaScript.

  • Function: Representa um bloco de código reutilizável que pode ser executado quando chamado.
